# Superstore Sales Analysis Dashboard - FUTURE_DS_01

## Project Overview
This project is a comprehensive Power BI dashboard developed as part of the Future Interns Data Science program. The dashboard analyzes the "Global Superstore" dataset to uncover key insights into sales performance, profitability, and operational efficiency.

## Key Insights (Business Highlights)
* **Total Revenue:** $2.30 Million
* **Total Profit:** $286.40 Thousand
* **Return Rate:** 5.91% (Critical metric for quality control)
* **Market Leader:** California is the top-performing state with over $457K in revenue.
* **Top Category:** Technology drives the highest revenue share.

## Dashboard Features
* **Executive Summary:** High-level KPI cards for Revenue, Profit, Units Sold, and Orders.
* **Temporal Analysis:** Monthly revenue and profit trends compared year-over-year.
* **Geographical Mapping:** A bubble map visualizing revenue distribution across the United States.
* **Product Performance:** Breakdown of revenue by Category and Sub-Category.
* **Customer Segmentation:** Analysis of sales across Consumer, Corporate, and Home Office segments.

## Tech Stack
* **Tool:** Power BI Desktop
* **Data Source:** Superstore Excel/CSV
* **DAX Measures:** Custom calculations for Profit Margin, Return Rate, and AOV.

## How to View
1. Download the `FUTURE_DS_01.pbix` file from this repository.
2. Open it using [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
